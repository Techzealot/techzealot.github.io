分布式事务

public enum TransactionType {
    
    LOCAL,//本地事务
  	XA, // XA事务,需数据库提供支持
  	BASE //柔性事务,如seata
}

JDBC规范本身并未提供事务管理器抽象，需自行实现事务管理器

JDBC使用事务的方式：

Connection conn = openConnection();
try {
    // 关闭自动提交:
    conn.setAutoCommit(false);
    // 执行多条SQL语句:
    insert(); update(); delete();
    // 提交事务:
    conn.commit();
} catch (SQLException e) {
    // 回滚事务:
    conn.rollback();
} finally {
    conn.setAutoCommit(true);
    conn.close();
}
